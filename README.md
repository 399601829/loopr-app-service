# README #

Backend for loopr iOS app.

## Environment variables
To run on local machine:

* RDS_HOSTNAME
* RDS_PORT
* RDS_DB_NAME
* RDS_USERNAME
* RDS_PASSWORD
```
========== RDS INFO ==========

RDS_PORT       : 5432
RDS_DB_NAME    : ebdb
RDS_USERNAME   : loopring
RDS_PASSWORD   : 
```

## How to start
```
gradle build
java -jar build/libs/loopring-ios-backend-0.1.0.jar
```
